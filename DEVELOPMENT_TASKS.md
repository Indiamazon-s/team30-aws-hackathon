# CultureChat 개발 태스크 목록

## 📋 프로젝트 개요
문화적 배려가 담긴 AI 번역 서비스 개발

### 🎯 핵심 기능
- **텍스트 번역**: 실시간 텍스트 번역 및 문화적 매너 분석
- **음성 번역**: AWS Transcribe를 활용한 STT 기능 + 텍스트 수정 가능
- **문화적 매너 체크**: 대상 국가 문화에 맞는 표현 분석 및 제안
- **광고 및 후원**: 사용량 기반 평가 요청 및 후원 안내
- **반응형 디자인**: 모바일 웹 환경 최적화
